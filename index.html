Q1
Let A = {1, 2, 3, 4} and B = {a, b, c}. Consider the relation f = {(1, a), (2, b), (3, c), (4, b)}.
Determine whether f is a function or not. Then find the range of f.
الحل:

1. هل العلاقة تمثل دالة (Function)؟
نعم، العلاقة f تمثل دالة. لأن كل عنصر في المجموعة المنطلق (Domain) A = {1, 2, 3, 4} يرتبط بعنصر واحد فقط وفقط في المجموعة المستقر (Codomain) B.

2. المدى (Range):
المدى هو مجموعة العناصر الفعالية في B التي استقبلت صوراً من A:

Range(f) = {a, b, c}
Q2
Determine whether the function f(x) = 2x + 5 from the set of integers to the set of integers is one-to-one. Explain your answer.
الحل:

نعم، الدالة هي One-to-One (Injective) متباينة.

التوضيح والإثبات الرياضي:
لإثبات أن الدالة متباينة، نفترض أن f(x₁) = f(x₂) ويجب أن نصل إلى أن x₁ = x₂:

Let f(x₁) = f(x₂)
2x₁ + 5 = 2x₂ + 5

Subtract 5 from both sides:
2x₁ = 2x₂

Divide by 2:
x₁ = x₂

بما أن x₁ = x₂ لكل العناصر في مجموعة الأعداد الصحيحة (𝐙)، فإن الدالة متباينة قطعاً.

Q3
Let A = {a, b, c, d} and B = {1, 2, 3, 4}. Suppose that f(a)=1, f(b)=2, f(c)=3, and f(d)=4.
State whether the function is injective, surjective, and bijective.
الحل:
  • Injective (One-to-One) / متباينة: نعم، لأن العناصر المختلفة في A لها صور مختلفة تماماً في B.
  • Surjective (Onto) / شاملة: نعم، لأن المدى (Range) يساوي تماماً المجال المقابل (Codomain B)، حيث تم تغطية جميع العناصر {1, 2, 3, 4}.
  • Bijective (One-to-One Correspondence) / تقابلية: نعم، بما أن الدالة متباينة (Injective) وشاملة (Surjective) في آن واحد، إذن الدالة هي Bijective.
Q4
Perform the following matrix addition:
A = [[2, 1], [3, 4]]     B = [[5, -1], [0, 2]]
Find A + B.
الحل:

عند جمع مصفوفتين، نقوم بجمع العناصر المتناظرة:

A + B = [
  [2 + 5,  1 + (-1)],
  [3 + 0,  4 + 2]
]

A + B = [
  [7, 0],
  [3, 6]
]
Q5
Find the transpose of the following matrix:
A = [[1, 2, 3], [4, 5, 6]]
الحل:

منقول المصفوفة (Transpose) يتم فيه تحويل الصفوف إلى أعمدة والأعمدة إلى صفوف:

المصفوفة الأصلية ذات حجم 2×3، والمنقول سيكون بحجم 3×2:

Aᵀ = [
  [1, 4],
  [2, 5],
  [3, 6]
]
